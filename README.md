🎮 Primeira versão do VALTÉCA!

✅ Jogo completo com sons e música ambiente
✅ Modo multiplayer (até 10 jogadores) e single player
✅ Frases geradas por IA via Ollama
✅ Botão de dica e controle de som

## Como jogar
1. Instale o Ollama (https://ollama.com/download/windows)
2. Baixe o modelo: `ollama pull gemma3:4b`
3. Baixe o VALTECA.exe abaixo
4. Dê dois cliques e divirta-se!

Sem Ollama? O jogo ainda funciona com frases prontas![VALTECA.zip](https://github.com/user-attachments/files/26169687/VALTECA.zip)
